# RAGBASE AI

AI-powered RAG chatbot built using:

- React + Vite
- Node.js + Express
- Gemini API
- Pinecone Vector Database
- LangChain

## Features

- PDF Question Answering
- Pinecone Vector Search
- Gemini AI Responses
- Modern Chat UI
- Typing Animation
- Responsive Design

## Tech Stack

Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express

AI:
- Gemini
- Pinecone
